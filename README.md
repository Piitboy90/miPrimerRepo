## PRESENTACIÓN
- Colección de 5 prompts profesionales aplicados a casos reales de negocio, 
construidos con técnicas de Prompt Engineering — delimitadores, output 
estructurado, lógica condicional y extracción de datos. Listos para 
integrarse en flujos de trabajo reales.
## TÉCNICAS USADAS
- Delimitadores para separar texto de instrucciones
- Output estructurado en JSON y HTML
- Audiencia y contexto específicos definidos
- Inferring y Summarizing aplicados a casos reales
- Lógica condicional para escenarios múltiples
- Límite de longitud en respuestas
