# PROMPT 02 – COPY PUBLICITARIO 

 

## SECTOR  

Marketing 

 

## PROBLEMA QUE RESUELVE 

Genera automáticamente copy publicitario persuasivo, para un producto de seguridad infantil, adaptado a una audiencia de padres primerizos, listo para usar en HTML sin necesidad de redactor.  

 

## PROMPT COMPLETO 
````
Instrucciones: 

Genera frase completa y correctamente estructurada 

Compuesta por 4 frases, con gancho poético al final. 

Silla de bebé de protección para coches de la marca Chicco, con las siguientes características, sistema de seguridad integrado, airbag en cabeceros, asiento acolchado para máximo confort (utiliza 3 términos sinónimos a lo descrito para integrarlo en la frase en armonía) 

Debe tener intención persuasiva para su venta 

Debe tener un lenguaje familiar y dirigido para padres primerizos. 

Devuelve el copy dentro de un <section> con un <h2> para el titular y un <p> para el cuerpo
````

## POR QUÉ FUNCIONA
Se aplican delimitadores para separar texto de instrucciones, se adapta un recurso de gancho final en la frase, se adapta al público específico, se limita la longitud y devuelve un copy publicitario listo en HTML.
